# Full-Stack-BookStore

This project is an e-commerce web application that provides user-interfaces for users to browse and purchase books.

# Pages

* Login/Register
* Home
* Favourites
* Cart
* Account
* Payment

# Used Technologies

* React for Front-End

* .Net 6 Web API for Back-End

* PostgreSQL for DataBase
